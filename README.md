# Restaurant / Takeaway Ordering System

## Overview

This project is a Full-Stack MERN application built for the **MAL3020 â€“ Full-Stack Development** module.  
It demonstrates a dynamic web system that allows users to browse menu items, place orders, and manage them in real time and admin to manage user oder.

## Features

- User registration and login system
- Browse and order menu items
- Real-time order status updates using WebSockets
- Admin panel for managing menu and orders
- MongoDB database for persistent storage
- Responsive front-end interface

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript (React)
- **Backend:** Node.js + Express.js
- **Database:** MongoDB
- **Testing:** Mocha & Chai
- **Containerization:** Docker
- **Version Control:** Git + GitHub
